## 當前局勢

Anthropic 正處於快速擴張與品質爭議並存的關鍵節點。Google 剛宣布高達 400 億美元的投資，為 Anthropic 注入充裕資金；但同時，Claude Code 近期的效能退步事件嚴重損傷了開發者信任，加上 Claude 4.7 Opus 的定價策略引發大量不滿。

Anthropic 首次正式承認工程疏失是導致 Claude Code 品質下滑的原因，這是官方對社群壓力的罕見正面回應。
