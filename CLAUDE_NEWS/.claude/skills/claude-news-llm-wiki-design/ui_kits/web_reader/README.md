# UI Kit — Web Reader

Public web rendering of the same Markdown corpus. Single-column reading at 720px prose width, sticky top nav, today feed on the homepage.

## Files
- `index.html` — today's digest as the homepage
- `ReaderShell.jsx` — top nav + footer
- `TodayFeed.jsx` — daily digest layout (5 sections)
- `EntityHeader.jsx` / `EntityList.jsx` — wiki views
- `SearchPalette.jsx` — ⌘K search
